# "Hot Tires...Hot Burgers!"

## This mod adds the Bigtop Burger food truck from Worthikid's YT series "Bigtop Burger", with some tweaks to the design.

### [CruiserSkin(v1.0.2)](https://thunderstore.io/c/lethal-company/p/EffMapis/CruiserSkin/) and [CruiserSkinNarPatch](https://thunderstore.io/c/lethal-company/p/NotARobotII/CruiserSkinNarPatch/) are already bundled with this mod so there's no need to install them. I do not have any plans to update this for CruiserSkin 1.1.0

##### An alternative version can be found [HERE](https://thunderstore.io/c/lethal-company/p/SpitfireSZN/BigtopCruiserPlus/), which includes some songs from the series that can play on the radio.
